# 🛡️ Malicious Node Detection in Networks Using Machine Learning

## Overview

This project presents a research study on detecting malicious nodes in computer networks using Machine Learning (ML) techniques. The primary objective is to identify malicious network behavior by training and evaluating multiple ML models and comparing their performance across various evaluation metrics.

## Objectives

* Detect malicious nodes in network traffic using supervised machine learning.
* Compare the performance of multiple ML algorithms.
* Evaluate models based on classification accuracy and other performance metrics.
* Identify the most effective model for malicious node detection.

## Methodology

The project follows a complete machine learning pipeline:

1. Data collection and preprocessing.
2. Feature engineering and selection.
3. Data splitting into training and testing sets.
4. Model training using multiple machine learning algorithms.
5. Performance evaluation and comparison.
6. Analysis of results and conclusions.

## Machine Learning Models

The following machine learning models were implemented and evaluated:

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)
* Naive Bayes
* Artificial Neural Network 

## Evaluation Metrics

The models were compared using several evaluation metrics, including:

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC
* Confusion Matrix


## Results

The experimental results demonstrate how different machine learning algorithms perform in detecting malicious nodes. A comparative analysis highlights the strengths and limitations of each model, helping determine the most effective approach for this classification task.
